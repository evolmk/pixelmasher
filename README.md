# Pixelmasher Portfolio App

Sails + Jade + Sass + Bootstrap

### Dependencies

Ruby 2.2.0
Postgresql

### Prerequisites

```bash
npm install sails -g
npm install -g foreman
```

### Setup

```bash
git https://github.com/evolmk/pixelmasher.git
cd pixelmasher
cp .env.example .env //then make database changes to .env
npm install
```

### Run

```bash
nf start
sails lift (without node foreman)
```
